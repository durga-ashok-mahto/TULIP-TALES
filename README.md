# 🌷 Tulip Tales

### Every Reader Deserves The Perfect Story.

Tulip Tales is an intelligent book recommendation and reading companion platform designed to help readers discover books that match their interests, goals, mood, and reading preferences.

The platform not only recommends books but also helps users track their reading journey, manage favorites, monitor progress, and gain insights into their reading habits.

---

## ✨ Features

### 🔐 Authentication
- User Registration
- User Login
- Secure Authentication Flow

### 📚 Personalized Book Recommendations
- Genre-Based Recommendations
- Mood-Based Recommendations
- Interest-Based Recommendations
- Goal-Based Recommendations
- Difficulty Level Matching
- Recommendation Score for each book

### 🎯 User Preferences

## 🧠 Recommendation Logic

Recommendations are generated based on:

- Genre Match
- Interest Match
- Reading Goal Match
- Mood Match
- Difficulty Match

Each recommended book is assigned a personalized recommendation score.
